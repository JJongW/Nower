## 커밋 타입 참고 / Commit Type Reference

- `feat`: 새로운 기능 추가 / Add new feature
- `fix`: 버그 수정 / Fix bug
- `refactor`: 코드 리팩토링 / Refactor code
- `style`: 코드 포맷팅, 세미콜론 누락 등 / Code formatting, missing semicolons, etc.
- `docs`: 문서 수정 / Documentation changes
- `test`: 테스트 추가/수정 / Add or modify tests
- `chore`: 빌드 업무 수정, 패키지 매니저 설정 등 / Build tasks, package manager config, etc.
