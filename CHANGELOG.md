## [0.1.0] - 2026-02-08

### Added
- 캘린더 월간/주간 뷰 기능
- 새 이벤트 추가 및 편집 기능
- Today 위젯 지원
- 캘린더 위젯 지원 (macOS)
- iCloud 동기화 기능
- 다크모드 완전 지원
- WCAG 접근성 기준 준수 (대비 4.5:1)
- UI 아이콘 리소스 (알람, 시간, 화살표)
- 시간대별 일정 정렬 기능
- 기간별 일정 표시 및 관리
- 반복 일정 기능
- 충돌 감지 및 해결 로직

### Changed
- 캘린더 UI/UX 개선
- 이벤트 테이블뷰 셀 디자인 개선
- 위젯 레이아웃 개선

### Fixed
- CloudSyncManager 동기화 안정성 개선
- 일정 입력 시 시간 입력 버그 수정
